THE APPLICATION IMPLEMENTS THE GREEN EVELATOR PROBLEM USING THE JAVA SOCKET API

Authors: Mahboobeh Abdal Mahmood Abadi AND Mehran Nasseri

INSTRUCTION TO RUN:
opent the command prompt and go to dist folder than:

java -jar elevators.jar -number <no of elevetors> -top <no of floors> -tcp

than

java -jar controler.jar

Note: wait for a few seconds (1 or 2 seconds) until controler prints out information about number of floors and elevators. Now you can start to test.
